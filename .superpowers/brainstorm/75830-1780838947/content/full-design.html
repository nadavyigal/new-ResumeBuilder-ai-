🚀 Growth — Three Parallel Engines

Engine 1: ASO
Start now · before approval
  • Title: "AI Resume Builder — ATS"
  • Keywords: resume builder, ATS resume, tailor resume, cover letter AI
  • Hebrew metadata variant: same listing, Hebrew keywords + description
  • Screenshots: show the before/after ATS score improvement
  • Preview video: 30 sec — paste job → see optimised resume
Compounds over time. First reviews critical for ranking velocity.
Engine 2: Free Web ATS Tool
Build sprint · post-approval
  • Landing page: "Free ATS Resume Checker"
  • No login required — paste resume text, get score + top 5 gaps
  • Result page CTA: "Fix it in Resumely" → App Store link (mobile-detect)
  • Hebrew version: same page, Hebrew UI + Israeli job market keywords
  • Attribution: ?ct=web-ats&at=organic on App Store link
Captures Google intent traffic. Highest-quality install source.
Engine 3: Founder + Community
Ongoing · starts now
  • LinkedIn: 1–2 posts/week — behind-the-scenes building, resume tips, success stories
  • Israeli market: post in Hebrew too; share in Israeli job-hunting Facebook groups
  • Goal for first 30 days: 20–50 motivated installs who will leave honest reviews
  • Each hired user = a LinkedIn post you share (with permission)
First reviews unlock ASO velocity. These users are your review base.
