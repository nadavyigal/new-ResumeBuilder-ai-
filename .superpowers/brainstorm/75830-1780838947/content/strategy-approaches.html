Growth + Pricing — 3 Approaches

All three accept the episodic model and use the same 4-tier pricing. The difference is the primary growth engine. Pick the one that fits where you want to spend your energy.

A

Volume flywheel — ASO + web tool + ambassador loop Recommended

Three engines running in parallel, each feeding the others. ASO brings organic installs. Web ATS tool brings SEO intent traffic. Success ambassador loop turns each hired user into social proof that powers both.

The flywheel:
ASO ranking → Install → Optimize + export → Got hired → LinkedIn badge → 5-star review → ASO ranking ↑
Google "ATS checker" → Free web score → "Fix it in Resumely" → Install

Why recommended

  • Each engine is independent — one slow week doesn't kill momentum
  • Ambassador loop is zero cost, just an in-app flow
  • LinkedIn success posts = credibility + founder content
  • Matches GTM plan channel rankings exactly

Requires parallel execution

  • Web ATS tool needs a build sprint
  • ASO optimisation needs copy + screenshot work before launch
B

Hebrew market wedge — dominate one market first

Lead with Israel. No competitor has strong Hebrew ASO, Hebrew-optimised UX, or Hebrew resume templates. Build a loyal first cohort in a market where you have a natural edge, then expand to English.

The wedge: Hebrew App Store metadata → Israeli job seekers find you first → strong early reviews in a low-competition niche → Hebrew LinkedIn + groups (Facebook job hunting groups in Israel are huge) → expand to English once proven

Why compelling

  • Blue ocean — no dominant resume app in Hebrew
  • Israeli job market is tech-heavy, high willingness to pay
  • Founder has direct community access

Limits scale early

  • Smaller total market than English
  • RTL PDF export risk (not verified yet)
  • Keeps you in one market for longer
C

Web-first funnel — SEO drives everything

Invest heavily in the free web ATS tool and programmatic SEO pages ("resume template for software engineers", "ATS resume for product managers"). Web traffic feeds app downloads at scale.

The funnel: 100+ SEO landing pages targeting job-title-specific resume searches → free web ATS score → "Get the full optimised version on Resumely" → App Store install

Highest ceiling

  • SEO compounds — content written once, traffic forever
  • Google intent traffic is the highest-quality install source
  • Web infrastructure already exists

Slow start + build-heavy

  • SEO takes 3–6 months to rank
  • Needs significant content/page creation work
  • Wrong focus for pre-launch stage
