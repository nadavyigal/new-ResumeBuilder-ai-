# Weekly Operator Checklist

## Monday
- Review KPI ladder and weekly deltas.
- Identify biggest drop-off in funnel.
- Pick one primary experiment for this week.
- Confirm analytics events are healthy.

## Tuesday
- Implement experiment changes (copy/UI/flow).
- QA EN and HE variants.
- Update tracking for new CTA/events.

## Wednesday
- Publish one high-intent content asset (EN + HE plan).
- Schedule social posts for both language tracks.
- Prepare newsletter draft.

## Thursday
- Distribute: LinkedIn, Reddit/X, community channels.
- Run engagement loop (comments/DM feedback).
- Capture objection language from users.

## Friday
- Compare experiment outcome vs hypothesis.
- Update learning log and prioritize next test.
- Refresh baseline metrics and claim validity status.

## Weekly Quality Gates
- No unverified numeric claims in public copy.
- No placeholder/garbled Hebrew text on funnel pages.
- Checker -> signup path works end to end in both locales.
- Pricing/checkout path tested in production-like environment.
