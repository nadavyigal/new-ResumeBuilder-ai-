## üí° Best Practices

### Content Marketing:
- Publish 2 blog posts per week minimum
- Focus on long-tail keywords (less competition)
- Include CTAs every 300-400 words
- Link internally between posts
- Update content quarterly

### Email Marketing:
- Personalize with first name
- Mobile-optimize all emails
- A/B test subject lines
- Send at optimal times (Tue-Thu, 10am-2pm)
- Clean list regularly (remove inactive)

### Social Media:
- Post 3-5x per week on LinkedIn
- Engage before promoting
- Use 3-5 relevant hashtags
- Reply to all comments within 24h
- Share user success stories

### SEO:
- Target 1-2 keywords per post
- Use keywords in H1, H2, first paragraph
- Add alt text to all images
- Build backlinks through guest posting
- Submit sitemap to Google

**Common Questions:**

**Q: Where do I publish the blog posts?**
A: If you have a `/blog` route in your Next.js app, add them there. Otherwise, set up a simple blog using MDX or a headless CMS like Contentful.

**Q: How do I create the graphics?**
A: Use Canva with the templates provided in `marketing/before-after-examples.md`. Export as PNG/JPG.

**Q: What email platform should I use?**
A: ConvertKit if you can afford $29/mo (best for creators). Otherwise, Mailchimp free tier (up to 500 subscribers).

**Q: Should I do all this myself?**
A: Start with what you can handle (1-2 channels), then expand. Don't try to do everything at once. Focus on quality over quantity.
